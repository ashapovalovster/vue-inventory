# Задача

## Макет интерфейса

```
+----------------------------+  +--------------------------+ 
|   +----------+ +-------+   |  |                          | 
|   |  Item    | |  Item |   |  |      SELECTED ITEM       | 
|   +----------+ +-------+   |  |                          | 
|       Selected: N/M        |  |                          | 
+----------------------------+  +--------------------------+ 

+----------------------------------------------------------+ 
| +--------------------------+  +--------------------------+  
| | [Item][Item][Item][Item] |  | [Item][Item][Item][Item] | 
| +--------------------------+  +--------------------------+ 
+----------------------------------------------------------+ 
```


## Описание блоков

- **Слева** - вещи пользователя
- **Справа** - вещи на выбор


### Блоки сверху:

- **Слева** - выбранные вещи из вещей пользователя (блок снизу-слева)
- **Справа** - выбранная вещь из вещей на выбор (блок снизу-справа)


## Объекты данных

Объекты вещей имеют **одинаковую структуру**, отличаются только **названием и номером**.

## Механика работы

### Блок снизу-справа:

- Можно выбрать **одновременно только 1 вещь**
- Выбранная вещь отображается **справа-сверху**


### Блок снизу-слева:

- Можно выбрать **от 1 до 6 вещей**
- Выбранные вещи отображаются в **верхней левой части в порядке выбора**
